ScrollPagination Class

Defined in: scroll-pagination.js:20
Module: scroll-pagination

A widget that makes a page or division can dynamically load data by scrolling mouse wheel without clicking any links. The UI pattern has varied namings such as Continuous Scrolling, Infinite Scroll, and Scroll Pagination.

_scroll = new Y.ScrollPagination({
    container: _node.one(".bd"),
    foldDistance: 500,
    delay: 1000,
    data: {
        source: "/welcome/q/get_notification_list?format=html",
        schema: {
            metaFields: {
                last_id: "last_id",
                last_ts: "last_ts",
                html   : "html"
            }
        },
        request: [
            "&last_id=" + _node.one("#last_id").get("value"),
            "&last_ts=" + _node.one("#last_ts").get("value"),
            "&r=" + parseInt(new Date().getTime(), 10)
        ].join("")
    }
});
_scroll.on("load", _loadHandler);

Attributes

autoLoad

Boolean

Defined in scroll-pagination.js:115

Set the autoLoad attribute to false and the UI will shows 'more' link. In our scenario we only want user has scrolling loading for 5 times. To achieve this, set this value during instance level.

Default: true

container

Defined in scroll-pagination.js:65

An optional attribute. Default we use body as container. If this is being set, it will be the ancestor node to append UI.

data

Object

Defined in scroll-pagination.js:163

A required attribute which helps this widget to set Y.DataSource.IO instance for you to load data. Currently this widget only support AJAX with JSON data. Check YUI DataSource to get more information.

data: {
    source: "ajax.php",
    schema: {
        metaFields: {
            count: "result.count",
            last_id: "result.last_id",
            last_timestamp: "result.last_timestamp",
            html: "result.html"
        }
    },
    request: "?page=1&r=" + parseInt(new Date().getTime(), 10)
}

delay

Number

Defined in scroll-pagination.js:88

The indicator shows loading immediately when user scrolls to the region. However, we delayed this time sending to server to avoid the performance issue when user scrolls faster than data loading.

Default: 500

foldDistance

Number

Defined in scroll-pagination.js:75

Distance below the fold for which data starts to load. Data doesn't start to load until the indicator is at most this distance away from (or above) the fold.

Default: 100

isEnd

Boolean

Defined in scroll-pagination.js:139

Set this value if you want stop loading data from server.

Default: false

node

Y.Node | String | HTML Element

Defined in scroll-pagination.js:102

The scrolling region. You may use window as scrolling region, but show control UI in another node. (container attribute helps.)

Default: window
